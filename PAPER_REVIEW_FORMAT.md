# 试卷 AI 命题分析（总评）固定模板

> 录入 agent 读透全卷后产出，写入 `biz_paper.remark`，前端 `MarkdownMath` 渲染。
> 🔴 **铁律**：① 结构固定四段、每张卷一致，不许各写各的；② **只写图表给不出的判断**——
> 难度/题型/章节分布下面已有图，**禁复述**；③ 内容精准（点到题号），不堆形容词；④ 每条带**加粗关键点标签**便于扫读。

## 模板（逐段填，标题原样保留）

```
## 定性
<一句话：这卷什么脾气 + 靠什么拉开档次。加粗 2 个关键词，≤40 字>

## 命题特点
- **<关键点标签>** — <一句说明，点到题号>
- **<关键点标签>** — <一句说明，点到题号>
（2–3 条，只写命题人意图/手法这类图表看不出的洞察）

## 高频易错
- **<错法标签>**：<题号 + 怎么错的>
（2–4 条，按失分概率排）

## 备课建议
- **整卷诊断**：<怎么用整卷分段查漏>
- **培优精选**：<抽哪几道做专项>
```

## 字段约束
- 四段标题**固定**：`定性` / `命题特点` / `高频易错` / `备课建议`。
- 每条以 `**标签** — 说明` 或 `**标签**：说明` 起头，标签是关键点。
- 全文控制在 ~400 字内（嘉兴卷基准 416 字）。
- **不出现**：难度星级分布、题型计数、章节分值占比（这些是下方图表的活）。
